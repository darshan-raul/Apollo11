## Critical Architecture Decisions (Do Not Revert)

1. **Frontend is NGINX-only on port 3000.** Three probe paths (`/healthz/live`, `/healthz/ready`, `/healthz/startup`) return 200 unconditionally. No supervisord, no separate Go probe server. The "supervisord + port 8085" decision in older notes is **wrong** — the actual code (Stage 4) is NGINX-only.

2. **DATABASE_URL `***` pattern** in docker-compose.yml is intentional — `***` is a literal string in YAML, password resolves correctly in container env (not a bug).

3. **Graceful SIGTERM**: Go services use `context.WithTimeout(30s)`. Python Identity uses `signal.signal()` (just logs, then uvicorn's built-in drain does the actual work). NGINX receives SIGTERM and exits within ~1s.

4. **Docs repo is Docusaurus** (not MkDocs). Build via `npm run build`, config in `docusaurus.config.ts`, custom CSS in `src/css/custom.css`, pages in `src/pages/`.

5. **No npm on host** — frontend builds happen inside Docker multi-stage (node builds → nginx serves).

6. **Helm chart bundles the access stack.** One `helm install` provisions the apps + StatefulSets + Envoy Gateway + MetalLB. Kustomize path is a plain-manifest subset (apps only) for dev-friendly iteration.
