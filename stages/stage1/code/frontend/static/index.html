Sidecar Pattern — Nginx

This frontend pod runs two containers:
1. Nginx — serves static assets (/static/), proxies API calls to Go
2. Go — handles API requests, serves the HTML shell

Pod: frontend
├── nginx (sidecar) :80
│ ├── /static/* → served directly (cache-friendly)
│ ├── /api/* → proxied to Go:3000
│ └── /health → proxied to Go:3000
└── frontend (Go) :3000
└── serves JSON API + HTML shell
